# Introduction to Machine Learning with Python

This book is an 8 chapter book for BS students in Data Analytics or Data Science.  It is very hands on with lots of coding examples.


**Author:** Dr. Ernesto Lee

## Chapters

1. Getting Started with Machine Learning and Python
2. Data Preparation and Exploratory Data Analysis
3. Supervised Learning: Regression
4. Supervised Learning: Classification
5. Model Evaluation and Validation
6. Ensemble Methods and Advanced Algorithms
7. Unsupervised Learning and Clustering
8. Machine Learning Pipeline and Real-World Projects

## Building Locally

```bash
# Install MyST
npm install -g mystmd

# Start development server
myst start

# Build for production
myst build --html
```
